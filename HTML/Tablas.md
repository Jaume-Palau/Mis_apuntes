# Documento para etiquetas de tablas.

## Etiquetas dentro de tabla: 

- table → Englobará todo el contenido de la tabla (bloque)

- caption → Indica en texto el contenido de la tabla

- thead → Es la cabecera. (Aporta semámtica)

- tbody → Es el cuerpo. Contendrá todos los datos de la tabla. (Aporta semámtica)

- tfoot → Siempre antes de tb. Sirve para acumulados, contabilizaciones o repetir la cabecera. (Aporta semámtica)

- th → Modela una celda de cabecera fila/columna. (Aporta semámtica)

- tr → Modela una fila de la tabla (bloque)

- td → Modela una celda común de la tabla 


## Atributos para celdas:

- colspan → sirve para fusionar celdas adyacentes por columnas

- rowspan → sirve para fusionar celdas adyacentes por filas
