### 1. Redes Feedforward (MLP)
👉 las más básicas  
datos tabulares (como tu dataset)  
flujo de datos en una sola dirección  

✔ las más usadas para empezar  

---

### 2. Redes Convolucionales (CNN)
👉 para datos espaciales  
imágenes, vídeo  

✔ detectan patrones (bordes, formas, etc.)  

---

### 3. Redes Recurrentes (RNN)
👉 para datos secuenciales  
texto, series temporales  

✔ tienen memoria del pasado  

---

### 4. LSTM / GRU
👉 variantes avanzadas de RNN  
mejoran la memoria a largo plazo  

✔ más estables que RNN básicas  

---

### 5. Transformers
👉 modelo más moderno para secuencias  
texto (ChatGPT, etc.)  

✔ muy potentes  
✔ sustituyen muchas veces a RNN  

---

### 6. Autoencoders
👉 para compresión de datos  
reducción de dimensionalidad  

✔ aprendizaje no supervisado  

---

### 7. GANs (Generative Adversarial Networks)
👉 para generar datos  
imágenes, caras, etc.  

✔ una red genera, otra evalúa  

---

### Resumen práctico (muy importante)

| Tipo de dato | Red recomendada |
|-------------|----------------|
| Datos tabulares | MLP (Feedforward) |
| Imágenes | CNN |
| Texto / secuencias | RNN / LSTM / Transformers |
| Generación de datos | GAN |
